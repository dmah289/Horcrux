Cách dùng — Easer

Điểm vào duy nhất là Easer.Evaluate(EaseType, t). Nó clamp t về [0,1] một lần rồi dispatch sang curve tương ứng.

float t = elapsed * inverseDuration;        // tránh phép chia trong hot path
float k = Easer.Evaluate(EaseType.OutBack, t);
transform.position = Vector3.LerpUnclamped(from, to, k);  // LerpUnclamped vì Back/Elastic overshoot
Lưu ý. Kết quả k có thể vượt [0,1] (Back/Elastic overshoot) → luôn dùng các hàm unclamped khi nội suy. Khuyến nghị xử lý điều kiện t >= 1 trước khi gọi để chốt đúng giá trị cuối.
